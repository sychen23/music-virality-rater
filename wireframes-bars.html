M + Pill Bars (number inside)

Rounded pill-shaped bars with percentage right-aligned inside the filled area.

M + Pill Bars (number outside)

Pill bars with label on left, bar in the middle, bold number on the right.

M + Segmented Bars

Battery/signal-style segmented blocks. Filled segments = score level.

N + Icon Badge Bars

Emoji in a rounded square badge, bar with gradient fill, number right-aligned.

N + Full Overlay Bars

Wide bars where label and number are overlaid on top of the fill. Reads like a status row.

N + Inline Label Bars

Subtle tinted background bar, emoji + name + mini bar + number all in one row.
